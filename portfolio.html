BOLLYFLIX

A Webpage which looks exactly like Netflix.

I've built a responsive clone of the Netflix homepage, showcasing my skills in modern frontend development. Using HTML for a robust structure and advanced CSS, I recreated the iconic user interface with a sleek design, dynamic movie carousels, and responsive layouts that adapt seamlessly to different screen sizes. This project highlights my ability to replicate complex, real-world designs and build a clean, scalable user interface from scratch.

While it looks like the real thing, it's just the tip of the iceberg. This is a static, frontend-only project that doesn't include any of the backend logic or data streaming functionality of the actual Netflix application.
